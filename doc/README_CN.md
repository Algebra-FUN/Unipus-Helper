## 简介
这是一个可以帮助你完成Unipus的谷歌插件，[更多细节详见英文版](../README.md)

## 功能
* 自动将答案填充入Unipus.
* 自动随机生成错误答案，以误导后台系统判断，增加真实性.
* ~~自动提交.~~(开发中)
* ~~每天自动挂机2小时.~~(开发中)

### 使用方法
* 在开发者模式下，在**Chrome拓展程序页面**，通过"加载已解压的拓展程序"进行安装即可
* 如果没有Chrome浏览器，可以使用其他**Chromium内核**的浏览器(如QQ浏览器)
